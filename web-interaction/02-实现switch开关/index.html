# 思路分享

使用元素模拟一些开关效果，几乎都需要先隐藏掉原生的样式，再自己编写样式。隐藏原生样式的方式有很多种，这里说一种比较简单的方式：使用 appearance 属性。

appearance 是一个 css 属性，意为”外观“。将它设置为 none 可以隐藏默认的样式，方便我们自定义样式。

这是默认的复选框：

为复选框设置了 appearance: none 后，我们在网页上就看不见它了，但它的”复选“功能依然有效，利用这一点，我们就可以轻松的实现 switch 样式，并且直接支持开关（即选中与取消选中）效果了。

# 代码实现

<input type="checkbox" class="switch" id="switch" />

.switch {
  /* 隐藏原生复选框的样式 */
  appearance: none;
  -webkit-appearance: none;
  -moz-appearance: none;
  -o-appearance: none;
  outline: none;

  /* 为复选框设置相对定位 */
  position: relative;

  /* 美化样式 */
  width: 84px;
  height: 34px;
  background-color: #e8e8e8;
  border-radius: 34px;
  padding: 4px;
  box-sizing: content-box;
  cursor: pointer;
  box-shadow: rgba(50, 50, 93, 0.25) 0px 30px 60px -12px inset, rgba(
        0,
        0,
        0,
        0.3
      ) 0px 18px 36px -18px inset;

  /* 设置过渡效果 */
  transition: all 0.3s;
}

.switch::before {
  content: "";
  position: absolute;
  left: 4px;
  width: 34px;
  height: 34px;
  border-radius: 50%;
  background-color: #fff;
  transition: all 0.3s;
  box-shadow: rgba(50, 50, 93, 0.25) 0px 2px 5px -1px, rgba(0, 0, 0, 0.3) 0px 1px
      3px -1px;
}

/* 复选框选中之后改变 switch 背景色 */
.switch:checked {
  background-color: #409eff;
}

/* 复选框选中之后改变开关位置 */
.switch:checked::before {
  left: 54px;
}

代码写完了，不算复杂，分享下我的思路：

  1. 编写一个复选框元素，去掉默认样式，美化样式，让他看起来像一个 switch 开关
  2. 为复选框设置相对定位，方便它的伪元素相对与复选框本身左右移动，模拟开关的效果
  3. 编写复选框的伪元素，美化样式，设置绝对定位，设置 left 让它默认（即复选框未选中状态）靠左展示
  4. 编写复选框选中状态的样式，当复选框被选中时，:checked 伪类对应的样式会生效，改变 switch 的背景色，同时将伪元素（小圆点）移动到 switch 的右边。

监听开关的改变，与监听复选框的改变一样：

const switchEl = document.getElementById("switch");
switchEl.onchange = function () {
  console.log(this.checked); // switch 开启打印 true，关闭打印 false
};
