# CSS

Cascading Style Sheet，级联样式表。它是一种计算机语言，是一种样式语言，但并不是编程语言或标记语言。

CSS 1994 年由哈肯·维姆莱和伯特·波斯设计，1996 发布 CSS1，1997 年 W3C 成立 CSS 工作组，接管了 CSS，并且在 1998 年发布了 CSS2。从 CSS3 开始，CSS 分为不同的模块来发布，比如 W3C 会单独发布关于（font）字体的样式标准，所以它并没有准确的发布日期。

# 语法规则

CSS 的语法规则主要由选择器和声明组成。

选择器用于指定需要应用样式的 HTML 元素，可以是元素的名称、类名、ID 等。例如，使用标签选择器 p 表示将样式应用于所有的<p> 元素，使用类选择器.class表示将样式应用于具有该类的元素。

以下是一个示例的 CSS 语法：

p {
  /* 声明由属性和值组成，用冒号分隔 */
  color: red;
  /* 每个声明以分号结束，最后一个样式的分号可以省略，但推荐加上 */
  font-size: 16px;
}

h1 {
  /* 多个声明可以组成一个声明组，用花括号括起来。 */
  color: blue;
  text-align: center;
}

为了提高可读性，可以每行只描述一个属性，并在每个声明后添加分号。

注释可以用来解释代码，并且不会被浏览器解析。CSS 注释以/*开始，以*/结束。

# 如何应用 CSS 到元素上

有三种方式可以将 CSS 样式应用到元素上：

  1. 内联样式（inline style）

    <p style="color: red;">这是一个使用内联样式的段落。</p>

  2. 内部样式表（internal style sheet），也叫做文档样式表（document style sheet）或内嵌样式表（embed style sheet）

    <!DOCTYPE html>
<html>
  <head>
    <style>
      p {
        color: blue;
      }
    </style>
  </head>
  <body>
    <p>这是一个使用内部样式表的段落。</p>
  </body>
</html>

  3. 外部样式表（external style sheet）

    <!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" type="text/css" href="styles.css" />
  </head>
  <body>
    <p>这是一个使用外部样式表的段落。</p>
  </body>
</html>

    /* styles.css */
p {
  color: green;
}

三种方式都很常用。

# link 元素

link 元素是外部资源链接元素，规范了当前文档和外部资源的关系。有点配置项的味道，通常情况 link 元素写在 head 元素中。

以下是 link 元素的一些常见用法：

链接样式表

使用 href 属性指定外部样式表的路径，并使用 rel 属性设置为"stylesheet"。例如：

<link href="main.css" rel="stylesheet" />

href 属性设置被链接资源的 URL，绝对路径和相对路径都支持。

链接站点图标

使用 rel 属性设置为"icon"，并使用 href 属性指定图标文件的路径。例如：

<link rel="icon" href="favicon.ico" />

rel 指定链接的类型open in new window，icon 表示站点图标，stylesheet 表示 CSS 样式。

链接其他资源

link 元素还可以用于链接其他类型的资源，如音频、视频等。具体使用方法取决于所链接资源的类型。

# CSS 颜色的表示方法

color 属性通常用来设置文本颜色，但实际上它设置的是“前景色”，比如当你为文本设置红色时，文本的下划线和中划线也会是红色。

# 颜色关键字

MDN: 颜色值open in new window

p {
  color: red; /* 红色 */
}

使用颜色的单词来设置颜色，单词不区分大小写。但这些单词不足以表示实际开发中用到的所有颜色。

# RGB 和 RGBA

# RGB

R（红）G（绿）B（蓝）三原色，通过控制三种颜色的不同数值，来控制最终的颜色。

p {
  background-color: rgb(255, 0, 0); /* 红色 */
}

rgb 函数中的值的取值可以是数字 0~255，也可以是百分比，255 对应 100%。

# RGBA

RGBA 的 A 是单词 alpha 的缩写，在 RGB 的末尾新增一个数值，表示颜色的透明度。

p {
  color: rgba(255, 0, 0, 0.5); /* 红色，透明度50% */
}

透明度的取值的范围是 0~1，或百分比，数字 1 表示 100%（完全不透明）。

透明度参数的整数部分的 0 可以省略，比如 rgba(255,0,0,0.5) 可以写成 rgba(255,0,0,.5)。

# 十六进制

rgb(,,) 是一个 css 函数，除了这种方式，也可以通过十六进制来表示 RGB 颜色：

p {
  color: #ff0000; /* 红色 */
}

将 rgb(255, 0, 0) 中的三个数值转换为 16 进制。头部的 # 是 CSS 中十六进制数的标识。

当十六进制颜色中某个元色的两位数相同时，可以缩写为一位数。比如 #FF0000 可以缩写为 #F00，#000000 缩写为 #000。

同样的，上面的 rgba(255,0,0,0.5) 可以表示为十六进制，即 #FF000080。

# 文本属性

# text-decoration

decoration 意为装饰，text-decorationopen in new window 可以设置文本的装饰线。

常见的取值：

  • none：无任何装饰线，常用于去除超链接元素默认的下划线
  • underline：下划线
  • overline：上划线
  • line-through：中划线（删除线）

# text-transform

transform 意为变形、变换。text-transformopen in new window 可设置文本内容的大小写。

常见的取值：

  • capitalize：每个单词的首字母大写
  • uppercase：所有字母为大写
  • lowercase：所有字母为小写
  • none：无作用

当文本中含有单词缩写、自定义的单词、拼音等非常规单词时，text-transform 的效果可能不理想，这时推荐使用 JavaScript 来实现。

# text-indent

text-indentopen in new window 设置文本的首行缩进。

常见的用法是设置首行缩进 2 个字符：

p {
  text-indent: 2em;
}

# text-align

text-alignopen in new window 用于设置子孙元素中的，行内元素（如文本）和行内块元素（如图片和输入框），相对于当前元素的水平对齐方式。

此属性对块级的子孙元素无效，除非你将块级元素的 display 修改为 inline 或 inline-block。

它的默认值是 start ，注意不是 left，在某些国家或地区，文本的方向是从右往左。

常见的取值：

  • left：内容左对齐。
  • right：内容右对齐。
  • center：内容居中对齐。
  • justify：将内容两端对齐，通过增加额外的空格来调整行内单词之间的间距。
<div class="container">
  <h1 class="left-align">Left Aligned Text</h1>
  <p class="left-align">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  </p>

  <h1 class="right-align">Right Aligned Text</h1>
  <p class="right-align">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  </p>

  <h1 class="center-align">Center Aligned Text</h1>
  <p class="center-align">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  </p>

  <h1 class="justify-align">Justify Aligned Text</h1>
  <p class="justify-align">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  </p>
</div>

.container {
  width: 300px;
  border: 1px solid black;
  padding: 10px;
}

.left-align {
  text-align: left;
}

.right-align {
  text-align: right;
}

.center-align {
  text-align: center;
}

.justify-align {
  text-align: justify;
}

前三种都没有什么疑问，最后一种 justify，会使最后一行以外的内容，两端贴边，剩下的空间均匀的分配给中间的单词，这会更利于阅读。这样的设计和网页出现的初衷有关（为了代替纸质报纸）。

如果想让最后一行也和其他行保持一致，可以设置 text-align-last 属性的值为 justify。

<div class="container">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
</div>

.container {
  width: 300px;
  border: 1px solid black;
  padding: 10px;
  text-align: justify;
  text-align-last: justify;
}

# letter-spacing 和 word-spacing

letter-spacing 可以控制英文字母之间的间距，对于中文字，则是每个字之间的距离。

word-spacing 控制单词之间的间距，浏览器以空格来判定单词，例如 Orange cat 会判定为两个单词，而 Orangecat 判定为一个单词。

<div class="container">
  <h1>Letter Spacing Example</h1>
  <p class="letter-spacing">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  </p>

  <h1>Word Spacing Example</h1>
  <p class="word-spacing">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  </p>
</div>

.container {
  width: 300px;
  border: 1px solid black;
  padding: 10px;
}

.letter-spacing {
  letter-spacing: 5px;
}

.word-spacing {
  word-spacing: 50px;
}

# 字体属性

# font-size

设置文字的大小。

常见的取值是数字+单位，比如 100px；也可以使用 em 单位（不推荐）；还可以设置百分比（基于父元素的 font-size 计算），1em 代表 100%，2em 对应 200%，50%表示父元素字体大小的一半。

实际上 font-size 支持多种取值，详见MDN: font-sizeopen in new window。

<div class="container">
  <h1 class="px-font">20px Font Size</h1>
  <p class="px-font">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  </p>

  <h1 class="em-font">1.5em Font Size</h1>
  <p class="em-font">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  </p>

  <h1 class="percent-font">150% Font Size</h1>
  <p class="percent-font">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  </p>
</div>

.container {
  width: 300px;
  border: 1px solid black;
  padding: 10px;
  font-size: 16px;
}

.px-font {
  font-size: 20px;
  /* 使用像素（px）作为单位，设置字体大小为20像素 */
}

.em-font {
  font-size: 1.5em;
  /* 使用em作为单位，相对于父元素字体大小的倍数，设置字体大小为父元素字体大小的1.5倍，也就是24px */
}

.percent-font {
  font-size: 150%;
  /* 使用百分比作为单位，相对于父元素字体大小的百分比，设置字体大小为父元素字体大小的150%，同样是24px */
}

通常情况下，浏览器的默认字体大小是 16px。

百分比是 css 中比较坑的一个东西，在不同情况下百分比的参照物也是不同的，后续会详细记录。

# font-family

设置文字的字体，可以设置一个或多个字体。

当浏览器渲染网页时，它会按照font-family 属性设置的字体顺序来选择要显示的字体。浏览器会逐个检查所指定的字体名称，如果计算机上存在该字体，则使用它来显示文本内容。如果计算机上没有找到第一个指定的字体，浏览器会继续检查下一个字体，直到找到一个可用的字体为止。

<div class="container">
  <h1>Font Family Example</h1>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
</div>

.container {
  width: 300px;
  border: 1px solid black;
  padding: 10px;
  font-family: 'Bauhaus 93', Arial, sans-serif;
}

如果电脑上安装了 Bauhaus 93 字体，那么展示效果如图，如果没有 Bauhaus 93 字体，浏览器会尝试下一个字体 Arial，以此例推。

设置字体时需要注意书写格式，当字体名称有空格时，需要使用引号（单双均可）将字体名称包括起来，多个字体之间用英文逗号分隔。

当指定font-family属性时，可以使用sans-serif 作为一个通用的字体族名称，以确保浏览器使用默认的无衬线字体来显示文本内容。如果计算机上没有指定的字体，浏览器将根据操作系统的设置选择合适的无衬线字体进行显示。

font-family 既可以设置计算机上安装的字体，也可以设置通过 @font-face 引入的外部字体（比如网络上的字体资源）。

<div class="container">
  <h1>Custom Font Example</h1>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
</div>

@font-face {
  /* 为引入的字体自定义一个名字，将在font-family中使用 */
  font-family: 'Bitstream Vera Serif Bold';
  /* 引入其他字体，可以是项目目录下的字体文件，也可以是互联网上的字体 */
  src: url('https://mdn.github.io/css-examples/web-fonts/VeraSeBd.ttf');
}

.container {
  width: 300px;
  border: 1px solid black;
  padding: 10px;
  font-family: 'Bitstream Vera Serif Bold', sans-serif;
}

# font-weight

设置字重，其实就是字体的粗细程度。

常见的取值：

  • 数值：1、100、200、...、900，1000；每一个数字表示一个重量。
  • normal：等于数值 400（默认值）
  • bold：等于数值 700（加粗效果）
<h1 class="normal-font">Normal Font Weight</h1>
<p class="normal-font">This is an example of normal font weight.</p>

<h1 class="bold-font">Bold Font Weight</h1>
<p class="bold-font">This is an example of bold font weight.</p>

<h1 class="custom-font">Custom Font Weight</h1>
<p class="custom-font">This is an example of custom font weight.</p>

.normal-font {
  font-weight: normal;
}

.bold-font {
  font-weight: bold;
}

.custom-font {
  font-weight: 600;
}

数值类型的值，1 和 1000 基本不会用到，如果设置为 123 或 123.45 等数值，某些浏览器可能不会精准划分，实际开发 400~700 用的最多。

strong、b、h1~h6 等元素默认的字重是 bold。

你可能会发现同样的字体，同样的字重，在不同的设备上显示的效果不一致（特别是 Android 设备和 iOS 设备），出现这种情况的原因一般有：

  1. 字体渲染引擎：Android 和 iOS 使用不同的字体渲染引擎，这些引擎可能对字体的呈现方式有微小的差异，例如字形的细节或锐利度。
  2. 屏幕分辨率和像素密度：Android 和 iOS 设备具有不同的屏幕分辨率和像素密度，这可能会影响字体的显示效果。在高分辨率屏幕上，字体可能会显示得更加清晰和锐利。

# font-style

设置字体的风格，如斜体。

<h1 class="normal-font">Normal Font Style</h1>
<p class="normal-font">This is an example of normal font style.</p>

<h1 class="italic-font">Italic Font Style</h1>
<p class="italic-font">This is an example of italic font style.</p>

<h1 class="oblique-font">Oblique Font Style</h1>
<p class="oblique-font">This is an example of oblique font style.</p>

.normal-font {
  font-style: normal;
}

.italic-font {
  font-style: italic;
}

.oblique-font {
  font-style: oblique;
}

italic 和 oblique 二者都可以让网页上的字体倾斜展示，它们有什么区别？

在字体设计中，一些字体家族会专门设计斜体字形，这些字形被称为italic。当使用具备italic 样式的字体时，浏览器会根据字体文件中的斜体字形来显示文本，从而呈现出真正的斜体效果。

然而，并非所有字体都提供了独立的italic样式。对于那些没有明确设计斜体字形的字体，使用italic样式并不会产生实际的倾斜效果，因为字体本身并没有设计斜体。

因此，如果一个字体没有明确的italic样式，使用italic并不会使文本倾斜。在这种情况下，可以考虑使用oblique 样式来对正常字体进行倾斜处理，以获得类似的视觉效果。

em、i、cite、address、var 等元素的默认 font-style 就是 italic。

# font-variant

指定字体的变体效果，例如小型大写字母，此属性不常用，看个乐子即可。

常见的取值有：

  1. normal：默认值，表示正常的字体变体效果。
  2. small-caps：将字母转换为小型大写字母形式。
<h1 class="normal-font">Normal Font Variant</h1>
<p class="normal-font">This is an example of normal font variant.</p>

<h1 class="small-caps-font">Small Caps Font Variant</h1>
<p class="small-caps-font">This is an example of small caps font variant.</p>

.normal-font {
  font-variant: normal;
}

.small-caps-font {
  font-variant: small-caps;
}

small-caps 可以将小写字母转换为大写字母，并保持小写字母的大小展示。原本就是大写字母的字符会被忽略。

# line-height

设置文本内容的行高，行高可以先简单理解为一行文字所占据的高度。

行高的存在是为了帮助人们阅读网页上的文本内容，行高太小，两行文本就会挤在一起，不便于人的阅读。

日常交流时会说“文字把 div 的高度给撑起来了”，这种说法不全对，其实撑起容器高度的并不是文字，而是文字的行高。

上面对行高的描述是不准确的，行高的严格定义是：两行文字，基线（baseline）之间的间距。

基线：与小写字母 x 底部对齐的线，也就是下图中的红线，图片来自维基百科open in new window

再来看中文的基线，图片来自互联网，这是两行相邻的文字：

🔍 既然两行文字基线之间的间距才是行高，为何行高的效果像是一行文字自己占据的高度？

先认识一个东西，叫做行距open in new window，指两行文字之间的空白部分（行间）。图中的 1、2、3、4 区域的高度是相等的。

并且两行文字，顶线到底线的距离也是相等的。

综合上面两个条件，才让我们在设置行高时，看起来像是在设置单行文本所占的高度。

行高除了让文字内容更便于阅读之外，还有个常见的用途就是让单行文本，垂直方向上居中对齐。

<div class="centered-text">Hello, World!</div>

.centered-text {
  width: 300px;
  height: 100px;
  /* 设置容器高度 */
  line-height: 100px;
  /* 设置行高等于容器高度 */
  text-align: center;
  /* 水平居中对齐 */
  border: 1px solid black;
  /* 添加边框以便观察 */
}

文字在一行中，会以顶线和底线为基准，垂直居中展示，假设字体大小为 20px，当我们设置行高为 40px 时，首先会分 20px 给字体展示，剩下的 20px，平均分成两份，一份 10px，分别给这行文字的上下各半个行距。

在 flex 等方便的布局方式出现之前，在 CSS 中实现元素的垂直方向上居中，是一件麻烦的事情。

# 通用选择器

通用选择器（Universal Selector），也叫做通配符选择器，能够选中页面上所有的元素。

某些前端项目中会出现这样的样式重置代码：

* {
  /* 去掉所有元素默认的内外边距 */
  margin: 0;
  padding: 0;
}

通配符选择器会影响所有的元素，通常认为它会消耗大量的性能，所以不推荐使用。如果真的需要重置 CSS 样式，建议将需要重置的元素手动列举出来。

/* 更推荐的做法 */
body,
h1,
h2,
h3,
h4,
h5,
h6,
p,
ul,
ol,
li {
  margin: 0;
  padding: 0;
  color: black;
}

# 元素、类、id 选择器

三个选择器都很常用，并且都很简单，简单说明即可。

元素选择器（type selectors），使用元素的名称来匹配：

/* 选中所有p元素 */
p {
  color: orange;
}

类选择器（class selectors），使用元素的类名来匹配：

/* 选中所有类名包含 error-text 的元素 */
.error-text {
  color: red;
  font-weight: 700;
}

id 选择器（id selectors），使用元素的 id 来匹配：

/* 选中 id 为 username 的元素 */
#username {
  font-weight: 600;
  color: #555;
}

id 如果由多个单词组成，单词之间可以使用中划线（-）、下划线（_）连接，也可以使用驼峰命名。

命名时最好避开元素名，最好不要这样：<div id="div"></div>

注意，在一个 HTML 文档中，id 的值应该是唯一的，即不能有两个或以上的元素的 id 是相同的。

# 进制

进制是一种计数方法，也称为进位计数制。在任何一种进制中，每一位上的数运算都是逢 X 进一位，其中 X 表示进制的基数。

最常见的进制是十进制，也就是我们平时使用的计数方法。在十进制中，每一位上的数运算是逢十进一。

除了十进制，还有其他常用的进制，如二进制、八进制和十六进制。

二进制是逢二进一，只包含 0 和 1 两个数字。

八进制是逢八进一，使用 0 到 7 这八个数字。

十六进制是逢十六进一，使用 0 到 9 以及 A 到 F 这十六个数字（A 表示 10，B 表示 11，依此类推）。

在 JavaScript 中，可以使用以下方式表示不同进制的数字：

  1. 二进制（Binary）：以0b或0B开头，后面跟着二进制数字。
let binaryNumber = 0b10 // 表示二进制数2
console.log(binaryNumber) // 输出: 2
  1. 八进制（Octal）：以0o或0O开头，后面跟着八进制数字。
let octalNumber = 0o10 // 表示八进制数8
console.log(octalNumber) // 输出: 8
  1. 十进制（Decimal）：直接使用数字表示。
let decimalNumber = 10 // 表示十进制数10
console.log(decimalNumber) // 输出: 10
  1. 十六进制（Hexadecimal）：以0x或0X开头，后面跟着十六进制数字。
let hexadecimalNumber = 0x10 // 表示十六进制数16
console.log(hexadecimalNumber) // 输出: 16

在 JavaScript 中，console.log()函数默认会将数字以十进制形式打印出来。无论使用哪种进制表示数字，它们在内部都会被转换为十进制数。如果希望在 console.log()中打印其他进制的数字，可以使用 toString()方法将其转换为指定进制的字符串形式，然后再进行打印。
